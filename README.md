** Steam on FreeBSD 11.0
 
 nvidia-driver or  xf86-video-ati
 
 pkg install doas 
 
 create /usr/local/etc/doas.conf :
 
 permit nopass keepenv yourusername 
 
 
 
 doas tar -zxvf steam.tar.gz  -C /
 
 steam

./install.sh
 
 ./steamrun
